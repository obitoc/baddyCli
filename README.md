# Usage
```sh
baddyCli -h
# Help
baddyCli -grep <fileName> <word>
# Grep Word from a File
baddyCli -ai <prompt>
#AI
baddyCli -info
# Device Info
```
----------
# Installtion
```sh
gh repo clone obitoc/baddyCli
# or
git clone https://www.github.com/obitoc/baddyCli
cd baddyCli
sh build.sh
```
